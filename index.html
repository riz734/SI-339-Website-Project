Hey! This is a website that captures my journey on a game called SimpleMMO. I started playing in April of 2020 and have been an active member of the game since then. This site showcases some of my achievements in game, as well as my collection of custom sprites and items.



When I started playing, I did not anticipate that I'd be playing the game for so long. As I started understanding more about the game and developed an interest in achieving certain goals, I fell in love with the game and its community.
